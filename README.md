Project description:
A modern, responsive React website designed for a professional car detailing center.
The platform showcases available services, pricing, and contact options, providing customers with a smooth and user-friendly experience.
The site includes interactive UI elements, service icons, a booking/contact form, and a clean layout tailored for automotive branding.
Built with React and optimized for speed, clarity, and easy navigation


Setup instructions:

npm install -g create-react-app-offline
npm install react-router-dom
crao -n app
cd app
npm start
git config --global user.name "xxxx"
git config --global user.email "xxxx@gmail.com"
